# Иконки плагина SNTZ

Здесь лежат ваши иконки. **Не перезаписывайте их** — используйте свои.

## Требования

| Файл | Назначение | Рекомендуемый размер |
|------|------------|----------------------|
| `plugin.png` | Иконка в панели Plugins | 48×48 или 96×96 px |
| `icon-dark.png` | Иконка панели (тёмная тема) | 23×23 или 46×46 px |
| `icon-light.png` | Иконка панели (светлая тема) | 23×23 или 46×46 px |

Формат: PNG (RGB или RGBA).

## Если иконки не отображаются

1. **Developer Mode:** при загрузке плагина выбирайте папку **UXP** (ту, где лежит `manifest.json`).
2. **Установленный .ccx:** пересоберите плагин в UXP Developer Tool (Export) и переустановите.
3. Полностью закройте Photoshop и откройте снова.
4. В UXP Developer Tool: Remove Plugin → Load Plugin (указать папку UXP).
