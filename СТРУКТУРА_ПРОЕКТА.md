# Структура SNTZ_API — пояснение файлов и папок

## Ответы на вопросы

### Нужна ли папка docs?
**Нет.** Папка `docs/` удалена. Вся документация — в `README.md` и `INSTALL.md` в корне. При установке через ComfyUI Manager пользователь получает **всё содержимое репозитория** — если бы docs была, она бы переносилась.

### Нужна ли папка PsUI?
**Да.** Она **переносится** при скачивании. В ней:
- `workflow_ps_linked_api.json` — шаблон workflow для ноды PS Linked (используется роутом `/sntz_ps_linked_config`);
- скрипты для Photoshop (export-layer-as-linked, set-comfyui-path и т.д.);
- `trigger_comfyui.py` — запуск ComfyUI из палитры PS.

Без неё нода SNTZ PS Linked Folder не сможет читать конфиг workflow.

### Где находится плагин Photoshop?
**Файл `.ccx`** — в корне `SNTZ_API/`. При клонировании пользователь получает `com.sntz.imagen-comfyui_PS.ccx` и устанавливает его двойным кликом.

## Файлы и папки (что остаётся в репозитории)

| Путь | Назначение |
|------|------------|
| `com.sntz.imagen-comfyui_PS.ccx` | Плагин Photoshop — двойной клик для установки |
| `__init__.py` | Точка входа плагина: регистрация нод, роут `/sntz_ps_linked_config` |
| `sntz_imagen.py` | Нода SNTZimage — генерация изображений через Gemini |
| `sntz_ps_linked.py` | Нода SNTZphotoshop — цикл Photoshop ↔ ComfyUI |
| `README.md` | Описание плагина и нод |
| `INSTALL.md` | Инструкция по установке (Manager, ручная) |
| `.gitignore` | Исключает ключи, `__pycache__`, `.DS_Store` |
| `web/` | Статика для ComfyUI (документация ноды в интерфейсе) |
| `web/docs/SNTZImagen.md` | Краткая справка по ноде SNTZimage (показывается в UI) |
| `PsUI/` | Скрипты и workflow для интеграции с Photoshop |
| `PsUI/workflow_ps_linked_api.json` | Шаблон workflow для PS Linked (читается роутом) |
| `PsUI/workflow_ps_linked.json` | Пример workflow в формате ComfyUI |
| `PsUI/trigger_comfyui.py` | Python-скрипт запуска ComfyUI из палитры PS |
| `PsUI/export-layer-as-linked-palette.jsx` | JSX для экспорта слоя в папку |
| `PsUI/set-comfyui-path.jsx` | Настройка пути к ComfyUI |
| `PsUI/assign-layer-id.jsx` | Вспомогательный скрипт |
| `PsUI/update-modified-content.jsx` | Обновление Linked SO |
| `PsUI/run_trigger.command`, `.bat` | Запуск trigger_comfyui |
| `PsUI/requirements_trigger.txt` | Зависимости для trigger_comfyui |
| `PsUI/README.md` | Инструкция по настройке PS |
| `PsUI/comfyui_url.txt`, `comfyui_path.txt` | Конфиг (пути) — создаются пользователем |
