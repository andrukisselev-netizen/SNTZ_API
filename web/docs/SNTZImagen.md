# SNTZimage

Генерация изображений по тексту (и текст + картинки) через SNTZ_API. Модели Gemini (gemini-2.5, gemini-3.1).

---

## Перед началом работы

**Настройки ComfyUI:**
1. **Settings** (⚙️) → **Comfy** → **Locale** → **Language** → **English**
2. **Settings** (⚙️) → **Experimental** → **Modern Node Design (Nodes 2.0)** — включить

**Общие напоминания:**
- ✓ VPN выключен
- ✓ API-ключ вставляется через Photoshop и один раз в каждую ноду

---

## Как использовать SNTZimage

1. **Выберите** соотношение сторон (`aspect_ratio`) и модель (`model`).
2. **debug** — оставьте **true**.
3. **credits_only** — оставьте **false** (для генерации).
4. Введите **prompt** и запустите Queue Prompt.

---

## Параметры

| Параметр | Описание |
|----------|----------|
| **prompt** | Текст описания изображения |
| **model** | gemini-2.5 (до 3 входных изображений), gemini-3.1 (до 5) |
| **api_key** | API-ключ (пусто = из .api_key) |
| **aspect_ratio** | 1:1, 16:9, 9:16, 4:3, 3:4 и др. |
| **resolution** | 1K, 2K, 4K |
| **debug** | true — логи в консоль |
| **credits_only** | false — генерация; true — только проверка баланса |

---

## Ограничения и рекомендации

- Не загружайте изображения разрешением больше 4K (следите за разрешением и DPI).
- Загружайте только JPEG и PNG.
- Желательно соблюдать соотношения сторон, поддерживаемые моделью (1:1, 16:9, 9:16, 4:3, 3:4 и др.).
- Модель может некорректно обрабатывать цветовые профили. Старайтесь использовать sRGB.

---

## Ошибки

При возникновении ошибок попробуйте несколько раз. Проверьте VPN, API-ключ, переключите модель. При повторении ошибки более 3 раз подряд обратитесь к системному администратору или в техподдержку.
